# Public Artifact Boundary

Public artifacts may include RuleOak Core, RuleOak Protocol, adapters, schemas, conformance fixtures, sanitized demos, and developer documentation.

Public artifacts must not include:

- private SafeDesk product source
- premium SafeDesk report templates
- payment or customer license logic
- signing keys or store credentials
- private beta user data
- cloud sync secrets
- real personal evidence files

SafeDesk is a private/commercial product powered by RuleOak Core. Only sanitized SafeDesk demos and public-facing documentation should appear in public repos.
