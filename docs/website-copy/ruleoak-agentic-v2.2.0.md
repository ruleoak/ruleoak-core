# Website copy: RuleOak Core v2.2.0 Developer Preview

## Hero

**Agent Firewall + Flight Recorder for AI agents**

Before your AI agent sends, deletes, spends, deploys, or changes production, RuleOak can block, approve, record, and replay the action.

## Three proof points

1. **Block risky actions before execution** — route tool calls through the Agent Firewall.
2. **Record every important action** — write evidence-grade JSONL with the Flight Recorder.
3. **Replay what happened** — generate timelines, incident reports, and local audit packets.

## Developer quickstart

```bash
npm install @ruleoak/core
npm run agentic:quickstart
```
