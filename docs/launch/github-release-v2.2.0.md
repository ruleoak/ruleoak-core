# RuleOak Core v2.2.0

Agent Firewall + Flight Recorder for AI agents.

Before an agent sends, deletes, spends, deploys, or changes production, RuleOak can block, approve, record, and replay the action.

## Try it

```bash
npm install
npm run agentic:quickstart
npm run agentic:public-demo
```
