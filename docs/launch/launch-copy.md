# Launch Copy

## One-liner

RuleOak adds governance to AI tool calls.

## Short description

RuleOak Core helps developers block dangerous tool calls, require approval for risky actions, keep policy outside prompts, and generate replayable evidence-backed audit records.

## README intro

```text
RuleOak Core is a local-first governance layer for AI tool calls. It sits between an agent and the tools it wants to use, evaluates policy before execution, records evidence, requests approval when needed, and generates audit-ready reports.
```
