# SafeDesk Demo Video Script

## Hook

AI agents can act on your computer. SafeDesk helps you stay in control.

## Demo

1. Choose a protected folder.
2. A mock AI assistant reads a safe note. SafeDesk allows it.
3. The assistant tries to delete a protected file. SafeDesk blocks it.
4. The assistant tries to send an email-like action. SafeDesk asks for approval.
5. SafeDesk records an evidence timeline.
6. The user exports a report.
