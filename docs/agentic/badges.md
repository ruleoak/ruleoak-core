# RuleOak badges

Use badges to communicate narrow compatibility claims.

```md
![RuleOak recorded](https://img.shields.io/badge/RuleOak-recorded-blue)
![RuleOak approval gated](https://img.shields.io/badge/RuleOak-approval--gated-green)
![RuleOak replayable](https://img.shields.io/badge/RuleOak-replayable-purple)
```

Allowed wording:

- RuleOak-compatible
- RuleOak-recorded
- RuleOak approval-gated
- RuleOak replayable

Disallowed wording unless separately evidenced:

- RuleOak certified
- compliance certified
- regulator approved
- security guaranteed
