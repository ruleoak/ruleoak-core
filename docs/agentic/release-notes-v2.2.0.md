# RuleOak Core v2.2.0 Release Notes

## Headline

RuleOak Core v2.2.0 is the stable developer release of **Agent Firewall + Flight Recorder for AI agents**.

## Highlights

- Stable `@ruleoak/core/agentic` export.
- Evidence JSONL v1.
- `.ruleoak.yml` v1.
- Agent Firewall, Flight Recorder, MCP Permission Gateway, approval, dry-run, replay, trust score, badges, and conformance kit.
- `ruleoak-py` v1.0.0 source package.
- RuleOak Agentic Skills v1.0.0 source package.

## Known limits

This release is a developer runtime, not a hosted enterprise control plane or certification product.
