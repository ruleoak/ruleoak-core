# What to Keep Private

Keep these private/commercial:

- full SafeDesk desktop/mobile UI.
- paid feature logic.
- customer/license logic.
- premium report templates.
- signed installer configuration.
- private roadmap.
- production backup/restore service.
- enterprise connectors.
- detailed vertical workflows beyond safe public demos.

Reason: RuleOak needs public credibility and commercial defensibility at the same time.
