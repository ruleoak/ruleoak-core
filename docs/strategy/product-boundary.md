# Product Boundary

RuleOak Core is the developer platform. SafeDesk is the consumer product.

## RuleOak Core

RuleOak Core provides Agent Firewall, Flight Recorder, evidence JSONL, approval gates, replay, guards, adapters, and conformance fixtures.

## SafeDesk

SafeDesk is a local-first consumer app that uses RuleOak Core to protect user files, ask before risky AI actions, and keep evidence of what happened.

## Vertical modes

Home Evidence, Creator Proof, Travel Proof, and Freelancer Proof are modes inside SafeDesk until user data proves they should be split into separate apps.
