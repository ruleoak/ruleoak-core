# Vertical Split Criteria

A SafeDesk vertical can become a standalone app only when most of these are true:

- 50+ real users tried that mode.
- 10+ users exported reports.
- 5+ users said they would pay.
- onboarding differs materially from SafeDesk core.
- pricing can stand alone.
- support burden justifies separation.
- website analytics show independent demand.

Until then, keep verticals as modes.
