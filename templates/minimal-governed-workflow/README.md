# Minimal Governed Workflow Template

Copy this template when you want to build a new RuleOak vertical workflow from a small starting point.

## Files

- `workflow.md` — describe the workflow goal and boundary.
- `policy.json` — define allowed, blocked, and approval-required actions.
- `sample-input.json` — replace with your own case/research/document input.

## RuleOak pattern

```text
policy → evidence → approval → audit
```
