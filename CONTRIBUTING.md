# Contributing

Public issues and pull requests are welcome. Maintainers are not obligated to merge pull requests, and external contributions are reviewed case by case.

## Contribution Policy

- Discuss large architecture changes before implementing them.
- Docs, examples, tests, and contract clarifications are welcome.
- Core security-sensitive changes receive stricter review.
- Keep local-first and explicit-boundary behavior intact.

## Developer Checks

```bash
npm install
npm run typecheck
```

## DCO-Style Certification

By contributing, you certify that you have the right to submit the contribution and agree that it is licensed under the repository license.
