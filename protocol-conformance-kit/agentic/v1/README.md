# RuleOak Agentic Conformance Kit v1

This MIT-licensed kit contains the stable protocol fixtures for external projects that want to emit RuleOak-compatible evidence without embedding the full RuleOak runtime.

Included:

- Evidence JSONL v1 fixture
- `.ruleoak.yml` v1 fixture
- expected validation behavior

Run from RuleOak Core:

```bash
npm run agentic:conformance
npm run agentic:conformance:json
```
