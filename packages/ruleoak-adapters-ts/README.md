# @ruleoak/adapters-ts v1.0.0

TypeScript adapters for using RuleOak as the permission, evidence, and replay layer in tool-calling agent applications.

Adapters are framework-style wrappers and do not require optional frameworks for tests. Use these with MCP-style tools, Vercel AI SDK-style tools, LangChain.js-style tools, OpenAI Agents JS-style tools, OpenClaw-style tools, and coding-agent command wrappers.
