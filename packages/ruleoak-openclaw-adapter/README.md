# @ruleoak/openclaw-adapter v1.0.0

OpenClaw-style permission and evidence adapter for RuleOak.

This is **an independent compatibility adapter, not endorsed by OpenClaw maintainers**. It is an optional adapter for OpenClaw-style action patterns: filesystem, shell, database, email, calendar, browser automation, skill install, and secret access.

It normalizes actions into RuleOak action envelopes, evaluates policy, records evidence, and fails closed for unknown dangerous actions.
