# @ruleoak/protocol v1.0.0

MIT-licensed protocol surface for RuleOak-compatible agent evidence.

This package is intentionally permissive. It lets third-party agent projects emit RuleOak-compatible evidence, manifests, policies, and badge claims without importing the AGPL RuleOak Core runtime.

Use this when you want compatibility. Use `@ruleoak/core` when you want the full Agent Firewall, Flight Recorder, approval, dry-run, replay, and enforcement runtime.
